

# **Financial Report – Power BI Dashboard**

This repository contains a concise and interactive **Power BI Financial Report** designed to analyze revenue, expenses, and profitability. The dashboard helps users quickly understand financial performance through clear visuals and KPI indicators.

## **Features**

* Revenue and expense breakdowns
* Profit & margin analysis
* Trend visuals (MoM, YoY)

## **File Included**

* `FINANCIAL REPORT BY ARKO (COMPLETELY BY ME).pbix` – Main Power BI dashboard

## **How to View**

1. Download the `.pbix` file
2. Open using **Power BI Desktop**
3. Interact with filters and visuals to explore insights
